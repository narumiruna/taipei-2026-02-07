## 📁 文件結構

各城市以下列 6 個檔案提供漸進式資訊揭露（Progressive Disclosure）：

```
gourmet/
  [城市名]/
    📌 overview.md    - 城市概要・策略・調查進度（5分鐘快讀）
    📥 inbox.md       - 未研究候選店暫存區（優先順序排定用）
    🏆 top-places.md  - 最終推薦清單 + Dining Strategy（預約從這裡開始）
    📊 candidates.md  - 全部候選店家一覽表（附分數・狀態）
    📝 notes.md       - 詳細調查筆記・證據資料（附資訊來源連結）
    🚫 excluded.md    - 排除店家與排除理由（稽核軌跡）
```

### 檔案使用說明
| 檔案 | 用途 | 適合讀者 |
|---------|------|--------|
| [overview.md](gourmet/taipei/overview.md) | 快速掌握城市特色與調查方針 | 初次查看者 |
| [inbox.md](gourmet/taipei/inbox.md) | 未研究候選店暫存、優先順序排定 | 調查者（工作管理用） |
| **[top-places.md](gourmet/taipei/top-places.md)** | **預約・決定拜訪地點** | **旅行者**（最重要） |
| [candidates.md](gourmet/taipei/candidates.md) | 總覽所有候選店、比較分數 | 規劃者、調查者 |
| [notes.md](gourmet/taipei/notes.md) | 確認證據資料、深入調查 | 想驗證調查正確性的人 |
| [excluded.md](gourmet/taipei/excluded.md) | 確認排除原因 | 稽核、需重新考慮時 |

## 🔍 調查方法論

本專案採用 **證據導向的調查手法**：

- ✅ **多來源交叉查核**: Google Maps、食べログ、Michelin Guide 等
- ✅ **50分標準化評分**: Taste/Quality, Value, Convenience, Consistency, Risk
- ✅ **可追溯性**: 所有推薦都附上資訊來源連結
- ✅ **透明度**: 包含排除理由在內的所有記錄

詳細調查流程: [AGENTS.md](AGENTS.md)（英文）
